Description

Data informs both operational decisions (discharge consents, incident response) and strategic planning (catchment management, regulatory reporting).

Typical Infrastructure

  • Fixed monitoring stations
  • Portable and automatic samplers
  • Telemetry hubs
  • Laboratory analysis facilities

Monitoring Methods

  • Chemical parameters (nutrients, metals, organics)
  • Biological indicators (invertebrates, algae)
  • Physical parameters (temperature, flow, turbidity)
  • Continuous and periodic sampling regimes

  • Pollution Event Detection
